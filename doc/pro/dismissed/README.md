# Dismissed

This folder stores ideas/plans you chose not to execute.

## Naming standard

- Use lowercase kebab-case.
- Prefer ending with `-plan.md`.
- Keep names short and topic-focused.

## Required header

Each dismissed file should include:

```md
- Status: dismissed
- Owner: <name>
- Started: YYYY-MM-DD | n/a
- Updated: YYYY-MM-DD
- Links: none | related docs/tests/PRs
```

## Required closure note

Include a short `## Dismissal Reason` section (1-3 lines).
