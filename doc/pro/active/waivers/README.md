# Waivers (Active)

This folder contains temporary policy waivers for currently active work.

## Files

- `waiver-register.md`: live register of active/expired/removed waivers.
- `waiver-register-template.md`: template for adding new waiver entries.

## Usage

1. Add new waiver entries to `waiver-register.md` using the template.
2. Keep owner, expiry date, and removal criteria mandatory.
3. Move or archive waiver records when related work moves out of `active/`.
