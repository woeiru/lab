# Waivers (Active)

This folder contains temporary policy waivers for currently active work.

## Files

- `*_waiver-register.md`: live register of active/expired/removed waivers.
- `*_waiver-register-template.md`: template for adding new waiver entries.

## Filename rule (required)

- Every file in this folder must use `yyyymmdd-hhmm_filename`.
- Use the file's last git touch time for the prefix.

## Usage

1. Add new waiver entries to `*_waiver-register.md` using the template.
2. Keep owner, expiry date, and removal criteria mandatory.
3. Move or archive waiver records when related work moves out of `active/`.
