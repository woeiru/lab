# Inbox

This folder stores ideas and plans not started yet.

## Naming standard

- Use lowercase kebab-case.
- End with one of: `-plan.md`, `-review.md`, `-followup.md`.
- Keep names short and topic-focused.

## Current items

- `function-library-review-plan.md`
- `validation-suite-failures-followup-plan.md`
- `git-history-cleanup-plan.md`
- `weekly-workflow-review-routine-plan.md`

## Required header

Each inbox file should include:

```md
- Status: inbox
- Owner: <name>
- Started: YYYY-MM-DD | n/a
- Updated: YYYY-MM-DD
- Links: none | related docs/tests/PRs
```
