# Queue (`doc/pro/queue`)

This folder contains triaged and prioritized work that is ready to start.

## Usage

- Move items from `doc/pro/inbox/` to `doc/pro/queue/` after triage.
- Keep this folder ordered by priority (or filename prefix).
- Move an item to `doc/pro/active/` when execution begins.

## Filename rule (required)

- Every file in `doc/pro/queue/` must use `yyyymmdd-hhmm_filename`.
- The prefix is the file's last git touch timestamp.
