# Queue (`wow/queue`)

This folder contains triaged and prioritized work that is ready to start.

## Usage

- Move items from `wow/inbox/` to `wow/queue/` after triage.
- Mandatory follow-ups from close/converge tasks may be created directly in
  `wow/queue/` when scope is clear and priority is already locked.
- Keep this folder ordered by priority (or filename prefix).
- Move an item to `wow/active/` when execution begins.

## Filename rule (required)

- Every non-README file in `wow/queue/` must use `yyyymmdd-hhmm_filename`.
- The prefix is the file creation timestamp; keep it stable after creation.
